#  LM2596 Buck Converter (DC-DC Step-Down Switching Regulator)

##  Project Overview
 LM2596-Based Buck Converter (5V Output) 
A step-down DC-DC converter designed using the LM2596 regulator IC to generate a stable 5V DC
output from a higher voltage source. Includes schematic, PCB layout, BOM, and Gerber files for
prototyping and embedded application development.

---

##  Objective

- Convert higher DC input (e.g., 12V / 24V) to regulated lower DC output
- Achieve higher efficiency compared to linear regulators
- Minimize ripple voltage
- Design a compact and optimized PCB layout

---

##  Key Components Used

- LM2596 Buck Regulator IC
- Inductor (Selected based on switching frequency & current rating)
-  Diodes (Fast recovery)
- Input Capacitor
- Output Capacitor
- Feedback Resistor Network
- PCB designed for switching applications

---

##  Design Approach

  - PCB layout optimized for:
   - Short switching loop
   - Proper grounding
   - Reduced EMI noise
   - Wide power traces

---

##  PCB Design Highlights

- Compact switching loop layout
- Thick traces for high current path
- Ground plane for noise reduction
- Proper placement of inductor and diode near IC
- Thermal considerations for heat dissipation
- DRC verified before manufacturing
