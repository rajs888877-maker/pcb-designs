#  2S Battery Management System (BMS) with Integrated Charger

##  Project Overview

This project involves the design and PCB implementation of a 2-Series (2S) Lithium-ion Battery Management System with integrated charging circuitry.

The system is designed to safely monitor, protect, and manage a 2-cell Li-ion battery pack while providing controlled charging functionality.

Complete schematic design, protection logic implementation, PCB routing, and manufacturing file generation were performed independently.

---

##  Objective

- Monitor individual cell voltages in a 2S battery pack
- Provide overcharge and over-discharge protection
- Implement short-circuit and overcurrent protection
- Integrate controlled charging circuitry
- Design compact and reliable PCB layout

---

##  System Features

- Individual cell voltage monitoring
- Overcharge protection
- Over-discharge protection
- Overcurrent protection
- Short-circuit protection
- Passive cell balancing (if implemented)
- Integrated charging control circuit
- High-current MOSFET switching path
- Proper filtering and decoupling network

---

##  Design Approach

1. Designed cell voltage sensing network for each battery cell.
2. Implemented protection logic for:
   - Overvoltage cutoff
   - Undervoltage cutoff
   - Overcurrent shutdown
3. Integrated MOSFET-based switching for charge/discharge control.
4. Designed charging stage ensuring controlled current flow.
5. Selected components based on:
   - Voltage rating
   - Current handling capacity
   - Thermal characteristics
6. Optimized PCB layout for:
   - High current handling
   - Low voltage drop
   - Reduced noise
   - Thermal stability
7. Generated Gerber files for manufacturing.

---

##  PCB Design Highlights

- Thick copper traces for battery current path
- Proper separation of control and power sections
- Optimized MOSFET placement for heat dissipation
- Clean ground routing strategy
- Compact and manufacturable layout
